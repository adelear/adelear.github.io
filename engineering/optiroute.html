ECE297 - OptiRoute

Geographic Information Systems · Pathfinding Algorithms · Graph Search · C++ · Data Structures

OptiRoute is our custom Geographic Information System (GIS) built entirely in C++. We developed the map implementation from scratch with a focus on usability, responsiveness, and scalability. The system includes robust pathfinding functionality, featuring both the A* algorithm and a greedy algorithm with simulated annealing to explore solutions to the Traveling Salesman Problem. Designed with a user-friendly interface and informed by usability testing, this project showcases how complex algorithms can be applied in a practical GIS environment while laying the groundwork for future extensions and applications.

Key Features

1) Learnable and Memorable User Interface

  • Intuitive Map View with full-screen display and clear street layouts
  • Zoom controls and “Clear Path” functionality
  • Search bars for intersections and points of interest (POIs)
  • On-screen guidance for directions and usage
  • POI categories: Essentials, Attractions, Food & Dining, Transportation
  • Accessible Help & Support button
  • Minimal and clear design with color-coded map elements

2) Live Weather and Temperature

  • Displays current temperature and conditions using real-time weather API
  • Provides environmental context for navigation and route planning

3) Contrast Mode

  • Enhances visibility by adjusting map color scheme
  • Improves accessibility in low-light environments
  • Ensures readability for visually impaired users

4) Pathfinding with A* Algorithm

  • Click two points to generate the shortest route with step-by-step directions
  • Route calculations consider travel time, turn angles, and penalties
  • The most efficient path is chosen based on total estimated time

User Testing

To evaluate usability, we conducted user testing sessions focusing on intuitive control, clear directions, loading speed, and accessibility. Participants interacted with the map and provided feedback through surveys. Results showed strong performance in intuitive control (4.39/5) and clear directions (4.35/5), with accessibility rated at 4.0/5. Loading speed was slightly lower (3.74/5), highlighting a key area for optimization. Overall, the system received an average score of 4.14/5.
